        * getElementsByClassName

        * Playing with style

        * From user

        * From server 
            * Sending a request
            * Handling data from a request
    * Asynchronous programming
        * Setting a timeout 
        * Old school callbacks
        * Creating a Promise
        * Async/await
        * Sending POST request
        * Async POST request
        * Reusing functions
        * Handling multiple asynchronous functions
        * Error handling
